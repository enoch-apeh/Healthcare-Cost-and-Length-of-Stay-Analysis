# Healthcare Cost and Length-of-Stay Analysis

This project analyzes a healthcare dataset in Python using a report-style Jupyter notebook.

## Project Files

- `healthdata.ipynb`: Main analysis notebook
- `healthcare_dataset.csv`: Source dataset used in the notebook
- `plots/`: Folder where chart images generated by the notebook are saved

## What the Notebook Covers

The notebook includes:

- data loading and quality checks
- data cleaning and feature engineering
- descriptive summary statistics
- demographic analysis
- condition frequency analysis
- billing analysis by condition, admission type, insurance provider, and test result
- length-of-stay analysis by condition, admission type, insurance provider, and test result
- medication frequency analysis
- monthly and yearly admission trend analysis
- a correlation heatmap for age, billing amount, and length of stay
- final findings and conclusion

## Visual Style

The charts are designed to be self-explanatory and use light shades of blue throughout for a consistent presentation. Category plots place categories on the x-axis and values on the y-axis.

## How to Run

1. Open `healthdata.ipynb` in Jupyter Notebook or JupyterLab.
2. Make sure `healthcare_dataset.csv` is in the same folder as the notebook.
3. Run the notebook from top to bottom.
4. Review the tables and charts generated by the notebook.
